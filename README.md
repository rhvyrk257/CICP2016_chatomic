# CICP_comic_annotation

##環境
pip install django  
pip install django-bootstrap-form


##動かす
1. makemigrations -- models.pyの変更を更新  
2. migrate -- データベースの設定を更新  
3. runserver -- サーバを起動

```html:sample
python manage.py makemigrations  
python manage.py migrate  
python manage.py runserver
```
